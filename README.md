# Auditing Dashboard

Digital transformation and compliance tracking platform.

**Live Demo:** https://auditing-dashboard-two.vercel.app/

## Run Locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Build

```bash
npm run build
```

## Tech Stack

- React + TypeScript + Vite
- Tailwind CSS
- Radix UI components

## Notes

- Data is static (no backend)
